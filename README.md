# 4DOF 3D Robotic Arm DH Simulation

Spatial 4DOF robotic arm simulation with DH kinematics, Jacobian-based inverse differential kinematics, simplified joint dynamics, and 3D visualization.

## Quick Start

```bash
pip install -r requirements.txt
python3 src/robot_arm_3d_demo.py --mode cartesian
python3 src/robot_arm_3d_demo.py --mode joint
python3 scripts/smoke_check.py
```
